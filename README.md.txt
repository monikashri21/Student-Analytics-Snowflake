# Student Analytics using Snowflake

## Project Overview
This project demonstrates how to use Snowflake SQL to load, manage, and analyze student performance data. The project performs SQL-based analysis and visualizes insights from a CSV dataset.

## Features
- Created a Snowflake database
- Created schema and table
- Created a stage and uploaded a CSV file
- Loaded data using the COPY INTO command
- Performed SQL analysis
- Created charts for data visualization

## Technologies Used
- Snowflake
- SQL
- CSV Dataset
- GitHub

## SQL Operations Performed
- CREATE DATABASE
- CREATE SCHEMA
- CREATE TABLE
- CREATE STAGE
- COPY INTO
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- COUNT
- AVG
- MAX
